# FrenchLibrary

Общее приложение-каталог для всей линейки French Learning Suite.

## Что делает

- Показывает все 7 учебных приложений на одной странице.
- Даёт быстрые ссылки на live-версии GitHub Pages.
- Даёт быстрые ссылки на исходные репозитории GitHub.
- Помогает использовать тренажёры как единую библиотеку.

## Содержимое

- `index.html` — стартовая страница-каталог.
- `README.md` — краткое описание проекта.
