Как использовать библиотеку

Эту страницу удобно публиковать отдельным репозиторием через GitHub Pages и использовать как общий вход во всю серию приложений.

Если нужно, дальше её легко расширить: добавить фильтры по темам, иконки, прогресс пользователя или единый бренд-блок для всех тренажёров.
